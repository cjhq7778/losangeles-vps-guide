## 九、常见问题 FAQ

**Q1：CN2 GIA 真的比普通线路快很多吗？**
对电信用户而言，晚高峰差别最明显。白天两者延迟可能只差 20-30ms，但晚上八九点普通线路开始丢包、延迟翻倍时，CN2 GIA 基本还能保持稳定。如果你是建站用途，这个差别直接体现在用户跳出率上。

**Q2：年付锁定会不会被商家跑路坑掉？**
任何年付都有这个风险，老牌商家也不能百分百保证。ZgoVPS 是 2021 年注册、2023 年开始在国内圈子活跃起来的，运营时间不算长但口碑积累还行。保守做法是先用季付试一两个月，确认线路稳定、客服响应正常再转年付。特价年付款本身价格低，损失上限可控。

**Q3：512MB 内存的入门款够干嘛？**
够跑个小博客（Typecho、Z-Blog、静态站）、做个轻量代理、放个监控脚本。WordPress 这种吃内存的别想了，1GB 起步比较稳。

**Q4：三网优化和双网优化价格差这么多，值吗？**
看你的访客结构。如果电信用户占大头，差价是值的；如果你自己只用联通或移动，访问者也主要是这两家，双网优化就够了。建站给全国人看的话，三网优化是基本盘。

**Q5：套餐流量用完了会怎样？**
ZgoVPS 套餐描述里写的是"Fair Use"（公平使用），超流量一般是限速而不是停机，具体限速幅度以官方政策为准。流量吃紧的话，选套餐时优先看流量档而不是带宽档。

**Q6：能不能换 IP？**
ZgoVPS 提供付费换 IP 服务，部分套餐有免费换 IP 机会（一般是新购两周内一次）。具体规则下单前在产品页看清楚或问客服。

**Q7：优惠码在哪里输入？**
